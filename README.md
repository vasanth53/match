# Match

A Progressive Web App (PWA) React Game

## Demo
[Live Demo](https://match-pwa.netlify.app/)

## Features
- Built with React
- PWA support (installable, offline-ready)
- Responsive design for mobile and desktop
- Fast and lightweight

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/vasanth53/match.git
   cd match
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Build for production:
   ```bash
   npm run build
   ```
